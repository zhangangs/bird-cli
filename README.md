# 自动化新建项目模块


## 安装方法

```
npm install bird-cli -g
```

## 使用方法

进入到相应的文件夹内。

```
bird init

//按照提示输入模块名称、模块描述和负责人即可使用。
...

```

### 会生成以下文件
```
//项目名称为：naturePlus

index.html   
naturePlus.controller.js
naturePlus.directive.js
naturePlus.factory.js
naturePlus.filter.js
naturePlus.module.js
naturePlus.scss
naturePlus.service.js
naturePlus.state.js
```
